3. 홈컨트롤러
- 메소드 추가
  - RequestMapping 설정
  - 메소드명 설정
  - viewName 설정(/member/signup)

- member 폴더 생성

6. MemberVO 추가
- vo패키지에 클래스 추가
- 필드는 DB 테이블의 속성들을 추가
- @Data를 추가
  - pom.xml에 롬복 의존성을 추가
- 생일의 세터를 추가
  - 문자열을 날짜로 변환하는 세터 추가

7. 홈컨트롤러에 메소드 추가
- 3번은 get이라면 6번은 post 메소드를 추가
- viewName을 설정(메인으로 리다이렉트)
- 화면에서 보낸 회원 정보를 메소드가 받아와서 확인
- 서비스에게 회원 정보를 주고 회원가입일 하라고 시킨 뒤 가입 여부를 알려 달라고 요청
- 가입이 되면 회원가입이 완료되었습니다를 알림 메세지로 보내고, 메인으로 리다이렉트
- 안되면 회원가입에 실패했습니다라고 알림 메세지를 출력하고 회원가입페이지로 리다이렉트
- 위에서 알림 메세지 출력 기능은 같이 구현

8. 멤버서비스에 메소드 추가/구현
- 전달받은 회원 정보를 이용하여 회원가입하는  코드를 구현
- 회원 정보 체크
- 문제가 없으면 다오에게 회원 정보를 주고 가입하라고 시킨 후 가입 여부를 정수로 요청
- 가입 여부가 0이 아니면 true를 0이면 false를 리턴

9. 멤버 다오/매퍼 메소드 추가/쿼리 구현
- insert문을 이용하여 추가
