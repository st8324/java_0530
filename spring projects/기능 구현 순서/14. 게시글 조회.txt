<게시글 조회>

1. url 선정 : /board/list

2. 링크 추가 : header.jsp
 - 게시글로 가는 링크 추가

3. 컨트롤러
 - 메소드 추가
  - value : /board/list
  - method : get
  - viewName : /board/list

4. list.jsp 추가

5. 컨트롤러
 - 3번 메소드에서 작업
 - 서비스에게 게시글 전체를 가져오라고 요청 한 후 가져온 게시글 리스트를 저장하여 화면에 전송

6. 서비스/임플
 - 메소드 추가 및 구현
 - 다오에게 게시글 리스트를 가져오라고 요청 한 후 가져온 게시글 리스트를 리턴

7. 다오/매퍼
 - select 문을 이용하여 게시글 리스트(전체)를 가져오는 쿼리를 작성
 - resultType 입력

8. list.jsp
 - 서버에서 보낸 게시글 리스트를 <c:forEach>와 <table>를 이용하면 화면에 출력
 - 제목은 <a>를 이용하여 링크를 추가 : /board/detail/게시글번호
