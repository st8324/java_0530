<게시글 검색>

1. list.jsp
 - 검색입력창과 버튼, 게시판을 선택할 수 있는 셀렉트 박스를 추가
 - form태그로 감쌈
   - action : /board/list
 - 입력창에 name을 지정
   - 검색어 : search
   - 게시판 : type

2. Criteria
 - 게시판과 검색어를 필드로 추가
 - 생성자에 초기값을 설정

3. 컨트롤러
 - 서비스에게 관리자 권한을 주면서 관리자 권한에 맞는 게시판 리스트를 가져오라고 요청
 - 가져온 게시판 리스트를 화면에 전송

4. 매퍼
 - 게시글 가져오는 쿼리문에 게시판과 검색어를 조회하도록 쿼리문 수정
 - 게시글 전체 수를 가져오는 쿼리문에 게시판과 검색어를 조회하도록 쿼리문 수정

5. list.jsp
 - 게시판 셀렉트 박스에 <c:forEach>를 이용하여 구성. 
 - 게시판 셀렉트 박스에 전체를 반드시 추가
 - 게시판과 검색어에 현재 선택된 게시판과 검색어가 유지되도록 구성
 - 페이지네이션 링크에 검색어와 게시판을 추가
