<게시글 삭제>

1. url 선정 : /board/delete/게시글번호

2. 링크설정 : detail.jsp
 - 게시글 상세에서 삭제 버튼을 추가
   - form태그로 감쌈
   - action : /board/delete/게시글번호
   - method : post
 - <c:if>를 이용하여 작성자만 보이도록 처리

3. 컨트롤러
 - 메소드 추가
  - value : /board/delete/게시글번호
  - method : post
  - viewName : /common/message
  - 게시글번호 확인 : @PathVariable
  - 로그인한 사용자 정보를 확인 
  - 서비스에게 게시글 번호와 회원 정보를 주면서 삭제하고 삭제 여부를 알려달라고 요청
  - 삭제 여부에 따라 msg, url 설정
    => msg : 게시글 삭제 성공!/ 실패!
    => url : /board/list

4. 서비스/임플
 - 메소드 추가
  - 회원정보 null체크
  - 다오에게 게시글 번호를 주면서 게시글을 가져오라고 요청
  - 게시글이 없으면 실패를 리턴
  - 게시글 작성자와 회원 아디를 비교하여 다르면 실패를 리턴

테스트

5. UploadFileUtils
 - 메소드를 추가
   - uploadPath와 파일위치가 주어지면 해당 파일을 서버에서 지우는 메소드를 추가(기존 프로젝트 코드 복붙)

6. 서비스/임플
 - 메소드 수정(4번에서 작업한 이후 )
  - 다오에게 게시글 번호를 주면서 첨부파일 리스트를 가져오라고 요청
  - 반복문을 이용
    - UploadFileUtils에 추가한 삭제 메소드로 첨부파일을 서버에서 삭제
    - 다오에게 첨부파일 번호를 주면서 첨부파일을 삭제하라고 요청

7. 다오/매퍼
 - 메소드 추가 및 쿼리 구현
  - delete문을 이용하여 첨부파일 삭제하는 쿼리 작성

테스트

   - 삭제 결과가 0이 아니면 참을 0이면 실패를 리턴
