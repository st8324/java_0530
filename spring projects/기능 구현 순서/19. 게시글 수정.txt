  - 가져온 게시판 리스트를 화면에 전송
  - 서비스에게 게시글 번호를 주면서 첨부파일 리스트를 가져오라고 요청

6. 서비스/임플
  - 메소드 추가
   - 다오에게 게시글 번호를 주면서 게시글을 가져오라고 요청 후 리턴

7. update.jsp
 - 서버에서 보낸 게시글 정보를 이용하여 화면에 구성
 - form태그
   - action : /board/update
   - method : post
   - enctype : multipart/form-data
 - 첨부파일을 a태그로 작성
   - x버튼을 추가해서 x버튼 클릭
     - input태그 file로 변함
     - input태그 hidden 삭제한 첨부파일 번호를 관리
     - name : fileNums
 - 게시판 리스트를 이용하여 게시판을 구성

테스트

8. 컨트롤러
 - 메소드 추가
   - value : /board/update/게시글번호
   - method : post
   - viewName : /common/message
 - 로그인한 사용자 확인
 - 수정할 게시글 정보 확인
 - 새로 추가된 첨부파일들을 매개변수로 추가
 - 삭제할 첨부파일 번호들을 매개변수로 추가
 - 서비스에게 게시글정보, 사용자정보, 첨부파일배열, 첨부파일번호 배열을 주고 게시글을 수정하라고 요청
 - 성공하면 msg에 "게시글 수정 성공!", 실패하면 "게시글 수정 실패!"를 저장하고 화면에 전송
 - url은 "/board/detail/게시글번호"로 설정하고 화면에 전송

9. 서비스/임플
 - 메소드 추가
 - 게시글 null체크
 - 사용자 null체크
 - 다오에게 게시글 번호를 주면서 게시글을 가져오라고 시킴
 - 가져온 게시글이 null이면 실패를 리턴
 - 게시글 작성자와 사용자 아이가 다르면 실패를 리턴
 - 다오에게 게시글 정보를 주면서 수정하고 요청
 - insertFileList를 이용하여 새로운 첨부파일 추가
 - 빈 첨부파일 리스트를 생성
 - 반복문을 이용하여 삭제할 첨부파일 번호 배열을 첨부파일 리스트로 변환
   - 첨부파일 번호를 꺼냄
   - 다오에게 첨부파일 번호를 주면서 첨부파일 정보를 가져오라고 요청
   - 첨부파일 리스트에 가져온 첨부파일을 추가
 - deleteFileList를 이용하여 첨부파일을 삭제
