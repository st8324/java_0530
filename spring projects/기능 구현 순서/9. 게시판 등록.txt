<게시판 등록>
1. url 설정 : /admin/board/type/insert

2. form태그에 action 설정 및 name 설정
 - boardTypeList.jsp에 tfoot 안에 있는 form태그의 action을 설정
- 각 입력 요소들에 있는 name을 설정

3. admin컨트롤러 메소드 추가 및 구현
 - value : /admin/board/type/insert
 - method : post
 - viewName : redirect:/admin/board/type/list
 - 화면에서 보낸 게시판 정보를 확인
 - 서비스에게 게시판 정보를 주면서 등록하라고 요청하고 등록 여부를 확인
 - 등록에 성공하면 새 게시판을 등록했습니다, 실패하면 새 게시판 등록에 실패했습니다를 알림메세지로 알려줌

4. admin서비스/임플 메소드 추가 및 구현
- 매개변수 체크
 - 타입이 일반 또는 이미지인지 확인
 - 읽기 권한이 0, 1, 9인지 확인
 - 쓰기 권한이 1, 9인지 확인
 - 게시판명이 중복됐는지 확인
   - 다오에게 게시판명을 주면서 일치하는 게시판 정보를 가져오라고 요청
   - 가져온 게시판 정보가 있으면 중복
   - 없으면 중복이 안됨
- 매개변수 체크에 이상이 없으면 다오에게 게시판 정보를 주면서 등록하라고 요청
- 등록되면 true를 안되면 false를 리턴

5. board다오/매퍼 메소드 추가및 쿼리 구현
- 메소드1 : 다오에게 게시판명을 주면서 일치하는 게시판 정보를 가져오라고 요청
  - select 태그를 이용하여 게시판명과 일치하는 게시판을 가져오는 쿼리문 구현
- 메소드2 : 다오에게 게시판 정보를 주면서 등록하라고 요청
  - insert 태그를 이용하여 게시판을 추가하는 쿼리 구현
